metaTitle: Text Area
metaDescription: Captures multi-line user input via a text area.
---

```jsx live=true
<TextArea placeholder="The quick brown fox jumped" />
```

## API Reference

This component is based on the `textarea` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

<ThemesPropsTable name="textArea" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Grid columns="3" align="center" gap="3">
  <TextArea size="1" placeholder="The quick brown fox jumped" />
  <TextArea size="2" placeholder="The quick brown fox jumped" />
  <TextArea size="3" placeholder="The quick brown fox jumped" />
</Grid>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Grid columns="2" gap="3">
  <TextArea variant="surface" placeholder="The quick brown fox jumped" />
  <TextArea variant="soft" placeholder="The quick brown fox jumped" />
</Grid>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Grid columns="2" gap="3">
  <TextArea color="gray" placeholder="The quick brown fox jumped" />
  <TextArea
    variant="soft"
    color="gray"
    placeholder="The quick brown fox jumped"
  />
</Grid>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Grid columns="5" gap="3">
  <TextArea radius="none" placeholder="The quick brown fox jumped" />
  <TextArea radius="small" placeholder="The quick brown fox jumped" />
  <TextArea radius="medium" placeholder="The quick brown fox jumped" />
  <TextArea radius="large" placeholder="The quick brown fox jumped" />
  <TextArea radius="full" placeholder="The quick brown fox jumped" />
</Grid>
```
