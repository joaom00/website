## API Reference

This component is based on the `button` element and supports [common margin props](/themes/docs/theme/layout#margin-props).

<ThemesPropsTable name="button" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3" align="center">
  <Button size="1" variant="soft">
    Button
  </Button>
  <Button size="2" variant="soft">
    Button
  </Button>
  <Button size="3" variant="soft">
    Button
  </Button>
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex gap="3" align="center">
  <Button variant="solid">Solid</Button>
  <Button variant="soft">Soft</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="surface">Surface</Button>
  <Button variant="ghost">Ghost</Button>
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Button color="crimson" variant="soft">
    Crimson
  </Button>
  <Button color="indigo" variant="soft">
    Indigo
  </Button>
  <Button color="grass" variant="soft">
    Grass
  </Button>
  <Button color="orange" variant="soft">
    Orange
  </Button>
</Flex>
```

### High-contrast

Use the `highContrast` prop to add contrast.

```jsx live=true
<Flex direction="column" gap="3">
  <Flex gap="3" align="center">
    <Button variant="solid">Solid</Button>
    <Button variant="soft">Soft</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="surface">Surface</Button>
    <Button variant="ghost">Ghost</Button>
  </Flex>
  <Flex gap="3" align="center">
    <Button variant="solid" highContrast>
      Solid
    </Button>
    <Button variant="soft" highContrast>
      Soft
    </Button>
    <Button variant="outline" highContrast>
      Outline
    </Button>
    <Button variant="surface" highContrast>
      Surface
    </Button>
    <Button variant="ghost" highContrast>
      Ghost
    </Button>
  </Flex>
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Button radius="none" variant="soft">
    None
  </Button>
  <Button radius="small" variant="soft">
    Small
  </Button>
  <Button radius="medium" variant="soft">
    Medium
  </Button>
  <Button radius="large" variant="soft">
    Large
  </Button>
  <Button radius="full" variant="soft">
    Full
  </Button>
</Flex>
```
