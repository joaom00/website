### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex gap="3">
  <Avatar variant="solid" fallback="A" />
  <Avatar variant="soft" fallback="A" />
</Flex>
```

### Color

Use the `color` prop to assign a specific [color](/themes/docs/theme/color), ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Avatar color="crimson" fallback="A" />
  <Avatar color="indigo" fallback="A" />
  <Avatar color="grass" fallback="A" />
  <Avatar color="orange" fallback="A" />
</Flex>
```

### High-contrast

Use the `highContrast` prop to add additional contrast.

```jsx live=true
<Flex direction="column" gap="3">
  <Flex gap="3">
    <Avatar variant="solid" fallback="A" />
    <Avatar variant="soft" fallback="A" />
  </Flex>
  <Flex gap="3">
    <Avatar variant="solid" fallback="A" highContrast />
    <Avatar variant="soft" fallback="A" highContrast />
  </Flex>
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Avatar radius="none" fallback="A" />
  <Avatar radius="small" fallback="A" />
  <Avatar radius="medium" fallback="A" />
  <Avatar radius="large" fallback="A" />
  <Avatar radius="full" fallback="A" />
</Flex>
```

      </svg>
