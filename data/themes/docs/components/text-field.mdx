metaTitle: Text Field
metaDescription: Captures user input via a text field.
---

```jsx live=true
<TextField.Input placeholder="The quick brown fox jumped" />
```

## API Reference

This component is based on the `input` element and supports [common margin props](/themes/docs/theme/layout#margin-props).

<ThemesPropsTable name="textField" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Grid columns="3" align="center" gap="3">
  <TextField.Input size="1" placeholder="The quick brown fox jumped" />
  <TextField.Input size="2" placeholder="The quick brown fox jumped" />
  <TextField.Input size="3" placeholder="The quick brown fox jumped" />
</Grid>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Grid columns="2" gap="3">
  <TextField.Input variant="surface" placeholder="The quick brown fox jumped" />
  <TextField.Input variant="soft" placeholder="The quick brown fox jumped" />
</Grid>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Grid columns="2" gap="3">
  <TextField.Input color="gray" placeholder="The quick brown fox jumped" />
  <TextField.Input
    variant="soft"
    color="gray"
    placeholder="The quick brown fox jumped"
  />
</Grid>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Grid columns="5" gap="3">
  <TextField.Input radius="none" placeholder="The quick brown fox jumped" />
  <TextField.Input radius="small" placeholder="The quick brown fox jumped" />
  <TextField.Input radius="medium" placeholder="The quick brown fox jumped" />
  <TextField.Input radius="large" placeholder="The quick brown fox jumped" />
  <TextField.Input radius="full" placeholder="The quick brown fox jumped" />
</Grid>
```
