## API Reference

This component inherits the props from the [Radix HoverCard](/primitives/docs/components/hover-card) primitive.

### Root

Contains all the parts of a hover card.

### Trigger

Wraps the link that will open the hover card.

### Content

Contains content to be rendered in the open hover card. This component is based on the `div` element.
