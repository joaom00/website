---
metaTitle: Select
metaDescription: List of options from which the user can select, triggered by a button.
---

```jsx live=true
<Select.Root defaultValue="apple">
  <Select.Trigger />
  <Select.Content>
    <Select.Group>
      <Select.Label>Fruits</Select.Label>
      <Select.Item value="orange">Orange</Select.Item>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="grapes" disabled>
        Grape
      </Select.Item>
    </Select.Group>

    <SelectSeparator />

    <Select.Group>
      <Select.Label>Vegetables</Select.Label>
      <Select.Item value="carrot">Carrot</Select.Item>
      <Select.Item value="potato">Potato</Select.Item>
    </Select.Group>
  </Select.Content>
</Select.Root>
```

## API Reference

### Root

Contains all the parts of a select. It inherits props from the Radix [Select.Root](/primitives/docs/components/select#root) part.

<ThemesPropsTable name="selectRoot" />

### Trigger

It inherits props from the Radix [Select.Trigger](/primitives/docs/components/select#trigger) and [Select.Value](/primitives/docs/components/select#value) parts. It supports [common margin props](/themes/docs/theme/layout#margin-props).

<ThemesPropsTable name="selectTrigger" />

### Content

It inherits props from the Radix [Select.Portal](/primitives/docs/components/select#portal) and [Select.Content](/primitives/docs/components/select#content) parts.

<ThemesPropsTable name="selectContent" />

### Item

The component that contains the select items. It inherits props from the Radix [Select.Item](/primitives/docs/components/select#item) part.

### Group

Used to group multiple items. It inherits props from the Radix [Select.Group](/primitives/docs/components/select#group) part. Use in conjunction with `Select.Label` to ensure good accessibility via automatic labelling.

### Label

Used to render the label of a group, it isn't focusable using arrow keys. It inherits props from the Radix [Select.Label](/primitives/docs/components/select#label) part.

### Separator

Used to visually separate items in the Select. It inherits props from the Radix [Select.Separator](/primitives/docs/components/select#separator) part.

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3" align="center">
  <Select.Root size="1" defaultValue="apple">
    <Select.Trigger />
    <Select.Content>
      <Select.Item value="orange">Orange</Select.Item>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="grape">Grape</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root size="2" defaultValue="apple">
    <Select.Trigger />
    <Select.Content>
      <Select.Item value="orange">Orange</Select.Item>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="grape">Grape</Select.Item>
    </Select.Content>
  </Select.Root>
</Flex>
```

### Variant

Use the `variant` prop on `Trigger` and `Content` to customize the visual style of the select.

```jsx live=true
<Flex gap="3" align="center">
  <Select.Root defaultValue="apple">
    <Select.Trigger variant="surface" />
    <Select.Content>
      <Select.Item value="orange">Orange</Select.Item>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="grape">Grape</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root defaultValue="apple">
    <Select.Trigger variant="soft" />
    <Select.Content variant="soft">
      <Select.Item value="orange">Orange</Select.Item>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="grape">Grape</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root defaultValue="apple">
    <Select.Trigger variant="ghost" />
    <Select.Content variant="soft">
      <Select.Item value="orange">Orange</Select.Item>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="grape">Grape</Select.Item>
    </Select.Content>
  </Select.Root>
</Flex>
```

### Color

Use the `color` prop on `Trigger` and `Content` to assign a specific color value, ignoring the global theme. Here we use a gray trigger with different colored contents.

```jsx live=true
<Flex gap="3">
  <Select.Root defaultValue="crimson">
    <Select.Trigger color="gray" />
    <Select.Content color="crimson">
      <Select.Item value="crimson">Crimson</Select.Item>
      <Select.Item value="other">Other</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root defaultValue="indigo">
    <Select.Trigger color="gray" />
    <Select.Content color="indigo">
      <Select.Item value="indigo">Indigo</Select.Item>
      <Select.Item value="other">Other</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root defaultValue="grass">
    <Select.Trigger color="gray" />
    <Select.Content color="grass">
      <Select.Item value="grass">Grass</Select.Item>
      <Select.Item value="other">Other</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root defaultValue="orange">
    <Select.Trigger color="gray" />
    <Select.Content color="orange">
      <Select.Item value="orange">Orange</Select.Item>
      <Select.Item value="other">Other</Select.Item>
    </Select.Content>
  </Select.Root>
</Flex>
```

### High-contrast

Use the `highContrast` prop on `Trigger` and `Content` to add contrast.

```jsx live=true
<Flex gap="3">
  <Select.Root defaultValue="default">
    <Select.Trigger />
    <Select.Content>
      <Select.Item value="default">Default</Select.Item>
      <Select.Item value="other">Other</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root defaultValue="default">
    <Select.Trigger highContrast />
    <Select.Content highContrast>
      <Select.Item value="default">High-contrast</Select.Item>
      <Select.Item value="other">Other</Select.Item>
    </Select.Content>
  </Select.Root>

  <Select.Root defaultValue="default">
    <Select.Trigger variant="soft" color="gray" highContrast />
    <Select.Content color="gray" highContrast>
      <Select.Item value="default">Gray high-contrast</Select.Item>
      <Select.Item value="other">Other</Select.Item>
    </Select.Content>
  </Select.Root>
</Flex>
```
