---
metaTitle: Icon Button
metaDescription: A button designed specifically for usage with a single icon.
---

```jsx live=true
<IconButton>
  <MagnifyingGlassIcon />
</IconButton>
```

## API Reference

This component is based on the `button` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

<ThemesPropsTable name="iconButton" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3" align="center">
  <IconButton size="1" variant="soft">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton size="2" variant="soft">
    <MagnifyingGlassIcon width="18" height="18" />
  </IconButton>
  <IconButton size="3" variant="soft">
    <MagnifyingGlassIcon width="22" height="22" />
  </IconButton>
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex gap="3">
  <IconButton variant="solid">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton variant="soft">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton variant="ghost">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton variant="outline">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton variant="surface">
    <MagnifyingGlassIcon />
  </IconButton>
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <IconButton color="crimson" variant="soft">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton color="indigo" variant="soft">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton color="grass" variant="soft">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton color="orange" variant="soft">
    <MagnifyingGlassIcon />
  </IconButton>
</Flex>
```

```jsx live=true
  <IconButton radius="large" variant="soft">
    <MagnifyingGlassIcon />
  </IconButton>
  <IconButton radius="full" variant="soft">

  </IconButton>
</Flex>
```
