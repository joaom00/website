## API Reference

This component inherits the props from the [Radix Radio Group](/primitives/docs/components/radio-group) primitive and supports [common margin props](/themes/docs/theme/layout#margin-props).

### Root

Contains all the parts of a radio group.

<ThemesPropsTable name="radioGroup" />

### Item

An item in the group that can be checked. An input will also render when used within a form to ensure events propagate correctly.
