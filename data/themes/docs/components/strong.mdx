---
metaTitle: Strong
metaDescription: Marks text to signify strong importance.
---

```jsx live=true
<Text>
  The most important thing to remember is, <Strong>stay positive</Strong>.
</Text>
```

## API Reference

This component is based on the `strong` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.
