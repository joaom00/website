---
metaTitle: Code
metaDescription: Marks text to signify a short fragment of computer code.
---

```jsx live=true
<Code>console.log()</Code>
```

## API Reference

This component is based on the `code` element and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

## Examples

### Size

Use the `size` prop to control the size of the text. The prop also provides correct line height and corrective letter spacingâ€”as text size increases, the relative line height and letter spacing decrease.

```jsx live=true
<Flex direction="column" gap="3" width="max-content" align="start">
  <Code size="9">console.log()</Code>
  <Code size="8">console.log()</Code>
  <Code size="7">console.log()</Code>
  <Code size="6">console.log()</Code>
  <Code size="5">console.log()</Code>
  <Code size="4">console.log()</Code>
  <Code size="3">console.log()</Code>
  <Code size="2">console.log()</Code>
  <Code size="1">console.log()</Code>
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex gap="3">
  <Code variant="solid">console.log()</Code>
  <Code variant="soft">console.log()</Code>
  <Code variant="outline">console.log()</Code>
  <Code variant="plain">console.log()</Code>
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Code color="crimson" variant="soft">
    Crimson
  </Code>
  <Code color="indigo" variant="soft">
    Indigo
  </Code>
  <Code color="grass" variant="soft">
    Grass
  </Code>
  <Code color="orange" variant="soft">
    Orange
  </Code>
</Flex>
```

### High-contrast

Use the `highContrast` prop to add contrast.

```jsx live=true
<Flex direction="column" gap="3">
  <Flex gap="3">
    <Code variant="solid">console.log()</Code>
    <Code variant="soft">console.log()</Code>
    <Code variant="outline">console.log()</Code>
    <Code variant="plain">console.log()</Code>
  </Flex>
  <Flex gap="3">
    <Code variant="solid" highContrast>
      console.log()
    </Code>
    <Code variant="soft" highContrast>
      console.log()
    </Code>
    <Code variant="outline" highContrast>
      console.log()
    </Code>
    <Code variant="plain" highContrast>
      console.log()
    </Code>
  </Flex>
</Flex>
```

### Weight

Use the `weight` prop to set the weight of the text. You can change the weight across breakpoints using the object syntax.

```jsx live="true"
<Flex direction="column" gap="2" align="start">
  <Code weight="normal">console.log()</Code>
  <Code weight="bold">console.log()</Code>
</Flex>
```
