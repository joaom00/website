---
metaTitle: Switch
metaDescription: A control that allows the user to toggle between checked and not checked.
---

```jsx live=true
<Flex gap="2">
  <Switch />
  <Switch defaultChecked />
</Flex>
```

## API Reference

This component inherits the props from the [Radix Switch](/primitives/docs/components/switch) primitive and supports common margin props `m`, `mx`, `my`, `ml`, `mr`, `mt`, `mb`.

<ThemesPropsTable name="switch" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3">
  <Switch size="1" />
  <Switch size="2" />
  <Switch size="3" />
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Switch color="crimson" defaultChecked />
  <Switch color="indigo" defaultChecked />
  <Switch color="grass" defaultChecked />
  <Switch color="orange" defaultChecked />
</Flex>
```

### High-contrast

Use the `highContrast` prop to add contrast.

```jsx live=true
<Flex gap="2">
  <Switch defaultChecked />
  <Switch defaultChecked highContrast />
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Switch radius="none" />
  <Switch radius="small" />
  <Switch radius="medium" />
  <Switch radius="large" />
  <Switch radius="full" />
</Flex>
```
