## Configuring your project

### 1. Import the CSS file

Import the global CSS file at the root of your application.

```ts
import '@radix-ui/themes/dist/index.css';
```

<Note>
  Note: This may vary depending of your framework of choice, but generally
  you'll want to import the CSS file at the root of your application.
</Note>

### 2. Add the Theme component

Wrap the root of your application in a `Theme` component.

```tsx
import { Theme } from '@radix-ui/themes';

export default function () {
  return (
    <Theme>
      <MyApp />
    </Theme>
  );
}
```

### 3. Start building

You are now ready to start using components and styles in your app:

```tsx
import { Flex, Text, Button } from '@radix-ui/themes';

export default function MyApp() {
  return (
    <Flex direction="column" gap="2">
      <Text>Hello from Radix Themes :)</Text>
      <Button>Let's go</Button>
    </Flex>
  );
}
```
